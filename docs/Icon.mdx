# 图标

## 引入方式

所有图标存放在主题包中，Muya UI 提供了默认的主题包`@muya-ui/theme-light`

```js
// 从默认主题中拿图标
import { AddIcon } from '@muya-ui/theme-light';

// 从自定义主题中拿图标，主题包需要单独安装
import { YourIcon } from '@qunhe/muya-theme-up';
```

## 简单用法

<Playground>
  <AddIcon />
</Playground>

## 设置颜色

默认会继承父级元素的颜色

<Playground>
  <div style={{ color: 'red' }}>
    <AddIcon />
  </div>
</Playground>

可以单独给 `Icon` 设置颜色，优先级更高

<Playground>
  <div style={{ color: 'red' }}>
    {/* 我是蓝色 */}
    <AddIcon color="blue" />
  </div>
</Playground>

## 尺寸

默认会使用父级元素的`font-size`作为元素的宽高

<Playground>
  <div style={{ fontSize: '30px' }}>
    <AddIcon />
  </div>
</Playground>

可以单独给 `Icon` 设置`fontSize`，优先级更高

<Playground>
  <div style={{ fontSize: '30px' }}>
    <AddIcon fontSize="14px" />
  </div>
</Playground>
