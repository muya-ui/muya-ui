# Anchor 锚点

可定位相关内容区域。是一种方便浏览内容的导航类组件。

## 用法

* 通常用于对于长内容，多区域的直线浏览。可用于实时定位切换。
* 一般一个页面中只会有一个锚点，不会出现2个。
* 区块内容少时可以用横向锚点，多时建议用纵向锚点。
