# toast 全局提示

用于全局告知用户操作反馈信息

## 用法

- 提供成功、失败、警告、提示、加载等反馈信息
- 用于不打断用户主线操作的轻量级提示，而非强制性信息
- 用于需要明确告知用户结果的提示
