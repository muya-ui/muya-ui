# Pagination 分页器

同类型内容长列表用分页面的方式承载，单次只加载一个页面。

## 使用场景

- 有大量同类型内容展现，加载内容需要较多时间，可使用分页器分解数据。
- 内容需要方便定位查找浏览时，可使用分页器。
