# Guide 新功能引导

新功能上线或新内容更新，需要一定程度的引导，引起用户关注，起到告知或强调的作用。

## 使用场景

根据功能类型不同，希望强调的程度不同，需采用不同程度的新功能引导，切莫对用户造成过度的干扰。
