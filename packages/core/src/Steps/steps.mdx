# Steps 步骤条

引导用户按照流程完成任务的导航条。

## 使用场景

当任务复杂或者存在先后关系时，可以将其分解成一系列步骤，从而简化任务，避免操作内容过多给用户造成心理压力。

## 使用原则

- 需展示一共具有多少步骤，并表明当前所处步骤
- 请勿进行步骤条嵌套或在同个页面中使用多个步骤条
