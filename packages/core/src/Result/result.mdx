# Result 结果页

用于反馈一些列任务操作、无数据状态的结果。

## 使用场景

- 当有重要操作需告知用户处理结果，且反馈内容较为复杂时使用
- 当目前没有数据时，用于用户提示
- 需要对结果有后续的引导
