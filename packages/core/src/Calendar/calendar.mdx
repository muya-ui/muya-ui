# Calendar 日历组件

按照日历形式展示数据的容器。

## 使用场景

当数据是日期或按照日期划分时，例如日程、课表、价格日历等，农历等。目前支持年/月切换。

## 组件

- `Calendar`: 单选日期。
- `RangeCalendar`: 区间选择。

{Demos}

### API

类型 `Dayjs` ， `ConfigType` 均来自 [dayjs](https://github.com/iamkun/dayjs)
