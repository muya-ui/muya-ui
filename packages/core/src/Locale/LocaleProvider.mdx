# LocaleProvider 国际化

## 使用场景

为组件内建文案提供统一的国际化支持。

目前组件已内置支持 [10 种语言](https://gitlab.qunhequnhe.com/muya-ui/muya-ui/tree/master/packages/muya-ui/src/Locale/resource)：简体中文、繁体中文、阿拉伯语、德语、英语、西班牙语、法语、日语、韩语、俄语。

## 时间处理

使用 [Dayjs](https://github.com/iamkun/dayjs) + [localizedFormat(plugin)](https://github.com/iamkun/dayjs/blob/dev/docs/en/Plugin.md#localizedformat)
