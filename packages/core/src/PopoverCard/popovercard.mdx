# PopoverCard 气泡卡片

点击、鼠标移入或聚焦元素，弹出气泡式的卡片浮层。

## 用法

当用户与被说明对象（文字，图片，输入框，按钮等）发生交互行为的 action 开始时, 即刻跟随动作出现一种辅助或帮助的提示信息，收纳到卡片中，根据用户的操作行为进行展现。

和 `Tooltip` 的区别是，用户可以对浮层上的元素进行操作，因此它可以承载更复杂的内容，比如链接或按钮等。

## 行为与状态

气泡卡片的交互行为分为：气泡出现、气泡收起、操作气泡中内容；

气泡卡片触发方式的方式有三种：悬停（hover）、聚焦（focus）、点击（click）；

**触发元素禁用状态下，也可触发气泡卡片；**
