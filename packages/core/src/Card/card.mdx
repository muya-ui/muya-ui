# Card 卡片

将信息聚合在卡片容器中展示。

## 使用场景

Card 组件可以将信息整合，形成自然区块，帮助界面信息模块感更强。可承载文字、列表、视频、图片等不同的内容。

## 组件

- `CardWrapper`: 卡片容器。
- `CheckBoxCard`: 附带 checkbox 组件的卡片。
- `CommonCard`: 卡片组件语法糖。
- `Card.Header`: 头部容器。
- `Card.Content`: 文案内容容器。
- `Card.Meta`: 文案组件语法糖。
- `Card.Actions`: 操作区容器。
