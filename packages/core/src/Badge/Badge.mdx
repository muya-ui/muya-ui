# Badge 徽标

出现在按钮、图标等元素旁的数字或状态标记。

## 使用场景

- 用于强调提示用户辅助信息，如消息/任务/订单/更新等。
- 用于显示需要处理的消息条数，通过醒目视觉形式吸引用户处理。
