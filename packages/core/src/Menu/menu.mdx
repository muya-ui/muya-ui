# Menu 导航菜单

为页面和功能提供导航的菜单列表。

## 使用场景

导航菜单是一个网站的灵魂，用户依赖导航在各个页面中进行跳转。一般分为顶部导航和侧边导航，顶部导航提供全局性的类目和功能，侧边导航提供多级结构来收纳和排列网站架构。

## 组件

导航菜单共含有以下五种组件：

- `Menu`： 菜单。
- `SubMenu`：子菜单。
- `MenuItem`：菜单项。
- `MenuItemGroup`：菜单项组。
- `MenuDivider`：菜单分割线。
