# Button 按钮

用于触发一个即时的操作，响应用户的需求。

## 使用场景

1. 主按钮：用于主行动点，一个操作区域只能有一个主按钮。
2. 默认按钮：用于非主要行动点，可与主按钮组合使用，也可以是没有主次之分的一组按钮。
3. 文本按钮：用于最次级行动点。
4. 链接按钮：用于外链行动点。
5. 图标：
   - 添加图标：用于对按钮含义补充解释，提高按钮识别效率。
   - 仅图标：需要在较小的空间内展示尽量多的按钮。

## 按钮属性

1. 危险：警示用户该操作存在风险；
2. 警告：提醒用户该操作存在风险；
3. 禁用：行动点不可用；
4. 加载中：用于异步操作等待反馈的时候，也可以避免多次提交；
5. 蒙层按钮：用于复杂的背景中，可灵活定制样式；

## 组件

按钮共含有以下五种组件：

- `Button`: 按钮。
- `InlineButton`: 文字按钮。
- `MaskButton`: 蒙层按钮。
- `IconButton`: 图标按钮
- `ButtonGroup`: 按钮组。
