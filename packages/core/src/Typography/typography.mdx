# Typography 排版

文本的基本格式。

## 使用场景

- 当需要展示标题、段落、列表内容时使用，如文章/博客/日志的文本样式。
- 当需要一列基于文本的基础操作时，如拷贝/省略/可编辑。

## 组件

共含有以下三种组件：

- `Typography.Title`：标题。
- `Typography.Paragraph`：段落。
- `Typography.Text`：文本。
