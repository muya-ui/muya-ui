# Breadcrumbs 面包屑

显示当前页面在系统层级结构中的位置，并能快速向上返回。

## 使用场景

- 当系统拥有超过两级及以上的层级结构时；
- 当需要告知用户【我在哪儿】时；
- 当需要向上导航的功能时。
