# Input 输入框

文本输入承载器，用于鼠标或键盘输入字符的最基础信息输入控件。

## 使用场景

- 需要用户输入表单域内容时。
- 提供组合型输入框，带搜索的输入框，还可以进行大小选择。

## 组件

- `Input`: 输入框。
- `RangeInput`: 区间输入框。
