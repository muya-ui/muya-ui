# Alert 全局警告

用于全局的展示重要的提示信息。

## 使用场景

- 提供成功、失败、警告、提示等反馈信息；
- 用于全局的、整个页面的提示，而非单一功能、模块的提示；
- 持续展示，用户可以主动关闭。
