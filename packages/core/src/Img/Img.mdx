# Img 图片

图片基础功能包含：

- 适配阿里云、金山云自动获取图片后缀
  - 阿里云：[OSS 图片处理文档](https://help.aliyun.com/document_detail/44686.html)
  - 金山云：[图片处理](https://docs.ksyun.com/documents/886)
- 图片懒加载

## 使用场景

在保留原生 img 的特性下，支持懒加载，自定义占位、加载失败等。

## 组件

- `Img`: 默认图片组件，别名 `ImgDiv` 、 `Picture`。
- `ImgSpan`: 设置 `component` = `span`。
- `ImgImg`: 设置 `component` = `img`。
- `ImgContainer`: 图片容器。
