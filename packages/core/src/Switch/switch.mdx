# Switch 开关

表示两种相互对立的状态间的切换开关选择器。

## 用法

需要表示开关状态/两种状态之间的切换时使用；和 `checkbox` 的区别是，切换 `switch` 会直接触发状态改变，而 `checkbox` 一般用于状态标记，需要和提交操作配合。
