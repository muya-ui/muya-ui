# Upload 上传组件

文件选择上传和拖拽上传控件。

## 用法

用于上传信息（图片、视频、模型等）到网页：

- 需要上传一个或者多个文件时
- 需要展现上传的进度时
- 需要使用拖拽交互时

## 构成

构成分为：触发器、上传结果。
