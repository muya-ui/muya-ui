# Popconfirm 气泡确认框

气泡式的确认框。和模态确认框相比交互形式更轻量。（气泡确认框也是一种类型的气泡卡片，因为使用比较高频，故单独作为一个组件存在）

## 用法

目标元素的操作需要用户进一步的确认时，在目标元素附近弹出浮层，询问或提示用户。
