# Carousel 走马灯

轮播器，是一组轮播的内容（图片、视频或其他内容）。

## 使用场景

- 当空间不足时，收纳一组时序上有顺序的内容。
- 走马灯轮播的内容可以是平级的图片，视频，卡片或者是多种内容的组合，他们需要保证内容不耦合。
